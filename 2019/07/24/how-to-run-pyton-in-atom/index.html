如何用atom直接运行python程序

Posted on 2019-07-24 |

现在有更好的方法了,此方法只能运行默认py解释器,请移步[https://mp.csdn.net/mdeditor/84959016#]查看新方法

导语

atom是一款非常好用的编辑器，但是他不能运行终端，所以让我们看看如何在atom上运行python程序吧。

  1. 打开你的atom编辑器
  2. 按组合键 ctrl+shift+p 输入 setting 回车或者点击file菜单选择setting
  3. 选择install 点击搜索框
  4. 搜索 atom-python-run 或 script
    前者（推荐）用F5启动程序，后者为组合键ctrl+shift+b
  5. 完成安装后，打开python文档，按对应组合键看看是不是已经弹出终端了。
