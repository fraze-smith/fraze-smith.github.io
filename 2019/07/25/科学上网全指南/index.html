科学上网全指南

形而上谓之道,形而下谓之器.

墙是什么

  • 防火长城是由我国著名的信息技术砖家-方滨兴院士主导建立的，为了天朝网民安全幸福上网的伟大国家科研项目。目前方滨兴砖家正在广州大学任教。
    （考研目标院校减一 -|-），如果你有意为这项丧尽天良的伟大事业添砖加瓦，可以选择报考方滨兴砖家的方班，希望他老人家不要翻墙来找我。
  • 维基：防火长城

    为何翻墙

    • 墙是天朝贵党维护社会的长治久安，保证人民的生命财产安全，并且抵制西方反动势力对伟大社会主义制度的渗透而建立的不可或缺的重要工具。但是墙有千般好，
      若为自由故，他妈的全可抛。

      翻墙之道

      修改hosts文件

      hosts是一个没有扩展名的系统文件，其基本作用就是将一些常用的网址域名与其对应的 IP 地址建立一个关联“ 数据库 ”。
      当用户在浏览器中输入一个需要登录的网址时，系统会首先自动从hosts文件中寻找对应的 IP 地址，一旦找到，系统就会立即打开对应网页，
      如果没有找到，则系统会将网址提交 DNS 域名解析服务器进行 IP 地址的解析。

这种方法在防火墙更换了审查方式后已经失效了，所以只简单介绍一下。

上网过程

  • 对于一般用户上网，首先打开游览器，然后输入网址（或者用网址导航，本质是一样的。），然后由DNS解析到对应的ip。然后网站响应，返回数据，
    游览器渲染。

但是如果你的DNS服务商是个大流氓，只要你访问他不喜欢的网站，他就给你瞎解析，让你到不了想去的ip。
这是选择一个优秀的DNS服务商就显得很重要。这里推荐google的DNS服务器：8.8.8.8(是不是很好记)。

修改hosts文件

  • MAC： 打开终端 输入 vim etc/.hosts 在第一行键入 8.8.8.8 就好了。
  • windows: 看这篇好了

    SSR + VPS （推荐）

    这是最稳定，也是俺最推荐的方式，这种方法需要一定的成本（大约每月三十块左右）。但这一点点成本是值得的，毕竟自由是无价。
    这个过程可能有一点复杂，但是完全不难，即使是小白也不需要担心。有任何问题也可以留言讨论。

SSR介绍

SSR是breakwa11在SS作者被约谈后停止对SS的更新后，发起的一个SS的分支项目。
不过在2017年被人肉后已经删库跑路了。
这里简单介绍下原理：当你发起的网络请求不是全部被拦截的，而是只拦截那些zf认为你不应该看的，而现在你通过ss服务向代理服务器发起请求，
进而代理服务器向目标网站发送请求，在接受到目标网站发送回的数据后，再将数据发送给你，当然这个过程是加密的，所以防火墙无法知道你发送和接受的
数据是什么内容，但是SSR也并不是完全安全的，参见这里
如果追求安全性，可以配合tor游览器一起使用。

SSR服务搭建

  1. 购买服务器

    1. 首先我们要选择一家VPS服务商，比较著名的有vultr，搬瓦工，还有一些台湾的服务商，就我自己使用的体验来看，vultr的服务是最稳定的，虽然价格比搬瓦工
      高一点，但是由于大多数人都选择搬瓦工搭梯子，所以现在ip已经被封的所剩无几了。
    2. 登陆vultr的官网，注册一个账号。最好使用谷歌邮箱，或者微软的。
    3. 然后登陆账号，点击左侧的billing进行充值，一般选择支付宝扫码支付就行，大约10刀需要70RMB左右。在完成充值后，返回主页.
    4. 点击右侧的加号，转到VPS购买界面,VPS地址最好选择日本，或者新加坡，延迟会低一点。系统的话选择自己熟悉的就好，如果没用过liunx系统，
      就选择centos。价格选择5刀每月的就好，如果觉得贵，新泽西还有3.5刀的，就搭梯子来说，都够用。现在点击deply就可以完成购买了。

  2. 服务器配置

    1. 操作系统安装需要一点时间，安装完成后，点击你的VPS查看ip和密码。
    2. 现在mac用户可以直接打开终端 输入 ssh root@VPS的ip 后输入前面你看到的密码连接服务器。
      windows用户需要下载一个ssh连接的工具，比如putty，点击前面的putty
      就可以直接下载，具体如何连接可以看这篇博客
    3. 在连接服务器后建议修改密码，以方便以后登陆维护 输入 passwd root 提示输入新密码。
    4. 现在安装ssr服务，这里我们使用秋水逸冰的一键安装。依次输入
      wget --no-check-certificate -O shadowsocks.sh https://raw.githubusercontent.com/teddysun/shadowsocks_install/master/shadowsocks.sh  
chmod +x shadowsocks.sh  
./shadowsocks.sh 2>&1 | tee shadowsocks.log
      这三条命令安装SS服务。这期间会依次提示你输入密码，端口和加密方式，如果你不熟悉的话，一直点回车就好。（加密方式选择7。ace-cfb-156）在完成安装后会有下面的提示：

      现在输入 vim /etc/shadowsocks.json 修改一下配置文件，只需要将server——ip的0。0。0。0改成你的VPS_ip即可。
    5. 输入 wget --no-check-certificate https://github.com/teddysun/across/raw/master/bbr.sh && chmod +x bbr.sh && ./bbr.sh 安装Google的BBR加速，
      可以有效的提高速度。
      现在服务器端的配置就完成了。又累了，改天再写

  3. 客服端下载

    • windows: 下载
    • mac os: 下载
    • android: 下载
    • ios: 下载

  4. 客户端配置
    客户端的配置其实都大同小异，需要填写的内容有1 服务器ip、2 端口、3 密码、4 加密方式、5 混淆一般不需要修改。这里以mac版为例。

    • 下载mac客户端后点击dmg文件安装即可。安装后打开软件，然后点击任务链的小飞机，选择servers，然后选择server perfercence。
    • 依次填入相应信息就好，最好对照着/etc/shadowsocks.json填写，注意：任何一项填错都不能连接。

  5. 最后
    由于天朝一直会对ip进行封禁，所以最好对服务器进行快照，再ip被封后直接再起一台服务器，恢复快照即可。

vpn gate

SoftEther VPN是一個由筑波大學研究生Daiyuu Nobori（登大遊）因碩士論文而開發的開源、跨平台、多重協定的虛擬私人網路方案，
此方案讓一些虛擬私人網路協定像是SSL VPN 、L2TP、IPsec、OpenVPN以及微軟SSTP都由同一個單一VPN伺服器提供。
在2014年1月4日以GNU通用公共授權條款的方式转变为开源软件。

这里是他们的官网,他们提供的教程挺详细的，配置也不困难。但是虽然软件是开源的，但代理服务是别人提供的，
所以安全性不能得到完全的保护。连接速度一般没有问题，适合有大量需求又不想花钱的人使用。

付费VPN

现在我所知道的VPN大多都被封禁，没有被封禁的也大多被国内公司收购（三成VPN服务商中资控股），安全性堪忧，极度不推荐使用。

tor游览器

Tor是实现匿名通信的自由软件。其名源於「The Onion Router」（洋蔥路由器）的英語縮寫。
用戶可透過Tor接達由全球志願者免費提供，包含7000+個中繼的覆盖网络，從而達至隱藏用戶真實地址、
避免網絡監控及流量分析的目的。Tor用戶的互聯網活動（包括瀏覽在線網站、帖子以及即時消息等通訊形式）相對較難追踪。
Tor的設計原意在於保障用戶的個人私隱，以及不受監控地進行秘密通信的自由和能力。

这里是他们的官网，tor游览器不需要配置，直接下载即可使用。下载完成后选择config，然后选择亚马逊的网桥。
点击connect，在漫长的等待后你应该就可以访问墙外世界了。 tor游览器的安全性非常高，但代价是速度特别慢，只能看文字类网页，不适合日常使用，
可以配合SSR一起使用，或者在所有翻墙方法都失效时，翻出去寻找原因，或寻找新方法。tor非常坚挺，即使在89三十周年其他方法全部失效时，依然可以使用。

I2P网络

I2P网路是由I2P路由器以大蒜路由方式组成的表层网路，建立于其上的应用程序可以安全匿名的相互通讯。
它可以同时使用UDP及TCP协议，支援UPnP映射。其应用包括匿名上网、聊天、撰写部落格和档案传输。

I2P网络是tor的升级版，使用方法差不多，但是我一直没用过，就不在这里介绍了。

赛风 自由门等（手机专用）

不推荐使用手机翻墙，安全性得不到保证，而且现在手机能用的VPN也是少之又少，我知道的只有赛风3还勉强能用，还时不时抽风。
